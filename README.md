# ML-Journey

This repo is compilation of my educational projects implemented during my self-study course. Projects and educational track is taken from [this Machine Learning Roadmap](https://github.com/justxor/MachineLearningRoadmap)

## 1. Python and Development Tools

1. **[l01-hello-cli](github.com/Anchaus/ml-journey/python-foundations/l01-hello-cli)**
   - **Description**: simple cli util that takes name and languages names as arguments and prints hello message using specified languages.
   - **Stack**: argparse, pytest
2. **[l02-csv-stats](github.com/Anchaus/ml-journey/l02-cli-stats)**
    - **Description**: cli util that takes name of csv file as an argument and prints it's basic stats by columns (type, missing values, min/max values and so on)
    - **Stack**: argparse, csv, pytest
